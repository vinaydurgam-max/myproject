# Local IoT Attack Detection (SQLite + Flask + React)

## Quick start
1. Backend:
   - Create and activate Python venv.
   - pip install -r backend_requirements.txt
   - cd backend and run: python app.py

2. Frontend:
   - npm install
   - npm run dev
   - Open http://localhost:5173

Backend API is available at /api/* and uses local database backend/database.db.
